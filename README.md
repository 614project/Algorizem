# Algorizem

알고리즘을 정리하기 위해 만든 라이브러리입니다.  
백준에서 특정 알고리즘의 코드를 빈번히 작성하는 시간을 아껴보고자 만들었습니다.

## 현재 구현된 알고리즘
(괄호는 클래스 이름)
- 데이크스트라 (Dijkstra) : 시작점으로부터 각 정점 별 최단거리를 구할수 있습니다.
- 애드몬드-카프 (EdmondsKarp) : 최대 유량을 구할수 있습니다.
- 분리집합(DisjointSet) : Union-Find 연산을 할수 있습니다.
- 세그먼트 트리(SegmentTreeNode) : 다양한 O(log n) 연산을 할수 있습니다.
- 비트 집합(BitSet) : 메모리 효율적인 boolean 컬렉션입니다.
- 에라토스테네스의 체(PrimeSieve) : 특정한 범위에서 소수를 가져오거나 판별할수 있습니다.

## 특징
구현해둘만한 요소만 구현되어 있습니다.  
즉, 세그먼트 트리를 예시로 트리가 생성되는 코드까지는 작성되어 있지만, 쿼리나 업데이트를 처리하는 매서드는 없습니다.  
이 프로젝트에 있는 모든 자료구조는 필요에 따라 수정하여 써야할수 있습니다.

## 기타
에라토스테네스의 체는 최적화가 적용되지 않았습니다. 곧 수정할 예정입니다.
