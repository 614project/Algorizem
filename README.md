# Algorizem

알고리즘을 정리하기 위해 만든 라이브러리입니다.  
백준에서 특정 알고리즘의 코드를 빈번히 작성하는 시간을 아껴보고자 만들었습니다.

## 현재 구현된 알고리즘
- 데이크스트라 (Dijkstra) : 시작점으로부터 각 정점 별 최단거리를 구할수 있습니다.
- 애드몬드-카프 (Edmonds-Karp) : 최대 유량을 구할수 있습니다.

## 구현할 예정
- 이분 매칭 : 두 그룹에 속한 정점끼리 최대한 많이 연결할수 있습니다.

## 특징
정점은 모두 uint 자료형을 사용합니다.
이외엔 long을 사용하였습니다. (데이크스트라의 경우 ulong을 사용합니다.)
